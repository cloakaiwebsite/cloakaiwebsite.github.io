1 Installation

Download & Install

Takes about 60 seconds. No account needed before download.

⊞ WINDOWS 10 / 11

Windows

  • Download CloakAI-Setup.exe
  • SmartScreen warning? Click More info → Run anyway
  • Follow the short installer
  • App launches automatically
macOS (Intel & M1/M2/M3)

Mac

  • Download CloakAI.dmg
  • Open DMG → drag to Applications
  • First launch: right-click → Open
  • Allow Microphone when macOS asks
Windows SmartScreen warning — normal

This appears for all unsigned apps. CloakAI is completely safe. Click More info → Run anyway. The warning only shows once on your machine.

macOS Gatekeeper — "Apple couldn't verify CloakAI"

This warning appears because CloakAI is not signed with an Apple Developer certificate. The app is completely safe. Use one of the methods below — try them in order.

A

Right-click method — try this first

Right-click (or Control+click) the CloakAI app → select Open from the menu → a different dialog appears with an Open button → click it. Done. This works on most Macs.

B

If method A didn't work — Privacy & Security

Double-click the app (it will be blocked) → open System Settings → click Privacy & Security in the sidebar → scroll down to the Security section → click Open Anyway → enter your Mac password. Works on macOS 13, 14 and 15.

C

If both above didn't work — Terminal command (macOS 15 Sequoia)

Open Terminal (press Cmd+Space → type Terminal → press Enter) and paste this command:

xattr -cr ~/Downloads/CloakAI.app

If you moved the app to Applications folder first, use: xattr -cr /Applications/CloakAI.app

Press Enter, then double-click the app — it opens normally.

4 Listening

Two Ways to Listen

Pick the mode that suits your interview. Switch anytime from the controls row.

⚡ RECOMMENDED

AUTO Mode

  • Flip the AUTO toggle ON
  • Fully hands-free — no buttons to press
  • Ignores noise, honks, background sounds
  • After 1.2s of silence sends to AI
  • Mic reopens after each answer
🎯 FULL CONTROL

Manual Mode (Alt key)

  • Turn AUTO off
  • Hold Alt while interviewer speaks
  • Release to get answer immediately
  • Best for noisy environments
  • Or use Start / Stop buttons
Which mode to use?

AUTO for most interviews — effortless. Use Manual (Alt) in noisy places or when you want precise control over what gets processed.

Set context before the interview starts. When asked "Tell me about yourself", CloakAI uses your context to give a role-specific, personal answer.

6 Screen Scan

Solve Anything on Your Screen

Interviewer shared a coding problem, MCQ, or diagram you can't copy? Screen Scan reads it and answers it.

1

Click "Screen Scan" or press Ctrl+Shift+A

CloakAI captures your full screen instantly — browser tabs, shared docs, anything visible.

2

AI detects what's on screen

Automatically identifies: coding problem, MCQ, buggy code, data structure, system design, or any written question.

3

Answer appears in the panel

Coding → solution + Big O. MCQ → answer + reason. Buggy code → bug list + fix. Any question → direct answer.

8 Troubleshooting

Common Issues & Fixes

Most problems are solved in one step. Start here before contacting support.

Short questions like "What is Docker?" aren't being detected +
Check that the correct microphone is selected in the mic pill. Speak at normal interview volume — the app starts recording the moment it detects sound. If still missing, increase your mic input volume in OS sound settings.
Random answers are appearing from background noise +
Switch to Manual mode using the Alt key. Nothing is processed unless you hold Alt. In a quieter room, AUTO mode works well — it filters brief sounds, honks, and anything without real English words.
Screen Scan not working on macOS +
Go to System Settings → Privacy & Security → Screen Recording. Find CloakAI and enable it. Restart the app after granting permission.
"License already activated on a different device" +
Each key works on one machine only. If you changed computers, email support@cloakai.in with your order ID and we will transfer the key for you.
The window is blank or invisible +
Press Ctrl+Shift+H — the window may just be hidden. If it stays blank after showing, close and reopen the app. On Windows, try turning off Hardware-accelerated GPU scheduling in Display Settings.
Answers are taking too long to appear +
CloakAI uses cloud AI (Whisper + GPT-4o). A slow internet connection under 5 Mbps will affect response time noticeably. On a good connection the first words of the answer start streaming within 1 to 2 seconds.
Can my interviewer see CloakAI? +
No. The app uses system-level APIs (WDA_EXCLUDEFROMCAPTURE on Windows, NSWindowSharingNone on Mac) that make the window impossible to capture — by Zoom, Teams, Meet, OBS, or even the Windows Snipping Tool. Only someone physically looking at your screen in the same room could see it.
Still having trouble? +
Email support@cloakai.in with your order ID if it is a license issue. We reply within 24 hours — usually much faster.
