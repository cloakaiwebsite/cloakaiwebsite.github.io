1 Installation

Download & Install

Takes about 60 seconds. No account needed before download.

⊞ WINDOWS

  • Download CloakAI-Setup.exe
  • If SmartScreen appears → More info → Run anyway

This appears for all unsigned apps. CloakAI is safe. Just click More info → Run anyway. The warning only appears once on your machine.

4 Listening

Two Ways to Listen

Pick the mode that suits your interview style. You can switch anytime.

⚡ RECOMMENDED

AUTO Mode

  • Flip the AUTO toggle ON
  • Sit back — CloakAI listens automatically
  • Detects when someone speaks, ignores silence
  • Smart filter: ignores noise, honks, background sounds
  • After 1.2 seconds of silence → sends to AI
  • Mic reopens instantly after each answer
🎯 FULL CONTROL

Manual Mode (ALT key)

  • Turn AUTO off
  • Hold Alt while the interviewer is speaking
  • Release Alt when they finish
  • Transcription + answer appear immediately
  • Or click Start Listening and Stop & Get Answer
  • Best for noisy environments
💡 Which mode to use?

AUTO for most interviews — effortless and hands-free. Manual (Alt key) when you're in a noisy place or want precise control over what gets processed.

6 Screen Scan

Solve Anything on Your Screen

When the interviewer shares a coding problem, MCQ, or buggy code — Screen Scan reads it and answers it.

1

Click "Screen Scan" or press Ctrl+Shift+A

CloakAI captures your full screen instantly — even content in browser tabs or shared documents.

2

AI analyzes what's on screen

It detects the type of content automatically — coding problem, MCQ, buggy code, raw data, system design, or any question.

3

Answer appears in the panel

Coding problem → optimal solution + Big O. MCQ → correct answer + explanation. Buggy code → bug list + fix. Any question → direct answer.

⚠ macOS only — grant Screen Recording permission

System Settings → Privacy & Security → Screen Recording → enable CloakAI. Only needed once.

8 Troubleshooting

Common Issues & Fixes

Most problems have a simple fix. Start here before emailing support.

Short questions like "What is Docker?" are not being detected +
Make sure your microphone is selected correctly (check the mic pill). Speak at a normal interview volume. The voice detection starts the moment sound is detected, so it should not miss short questions. If it still misses them, try increasing your mic input volume in your OS sound settings.
Random answers appearing from background noise / honks +
Switch to Manual mode with the Alt key. This gives you 100% control — nothing is processed unless you are holding Alt. In a quieter environment, AUTO mode works well. The smart filter ignores brief sounds under 60ms and sounds with no recognizable English words.
Screen Scan not working on macOS +
Go to System Settings → Privacy & Security → Screen Recording. Find CloakAI and enable it. Restart CloakAI after granting the permission.
"License already activated on a different device" +
Each license is bound to one machine. If you changed computers, email support.cloakai@gmail.com with your order ID and we will transfer your key.
The window appears blank or completely transparent +
Press Ctrl+Shift+H — the window may just be hidden. If it's genuinely blank, close and reopen CloakAI. On Windows, try disabling Hardware-accelerated GPU scheduling in Display Settings if the issue persists.
Answers are slow to appear +
CloakAI uses cloud APIs (Whisper + GPT-4o). A slow internet connection (under 5 Mbps) will noticeably affect speed. Check your connection. On a good connection, the first word of the answer appears within 1–2 seconds of transcription completing.
Will my interviewer see CloakAI? +
No. CloakAI uses a Windows system API (WDA_EXCLUDEFROMCAPTURE) and macOS NSWindowSharingNone that make the window completely uncapturable — by Zoom, Teams, Meet, OBS, and even the Windows Snipping Tool. The only way someone could see it is by physically looking at your screen in the same room.
Still having issues? +
Email us at support.cloakai@gmail.com. Include your order ID if it's a license issue. We reply within 24 hours — usually much faster.
